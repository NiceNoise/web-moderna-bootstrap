La Web Moderna

HTML

HTML (HyperText Markup Language) es el lenguaje estándar usado para crear y estructurar páginas web. Funciona como el esqueleto de cualquier sitio web, permitiendo organizar el contenido mediante un sistema de etiquetas (tags) que el navegador interpreta para mostrar la información al usuario.

CSS

CSS (Cascading Style Sheets) Hoja de Estilos en Cascada es un lenguaje de diseño que se utiliza para definir la presentación visual de documentos HTML. Si HTML es el esqueleto, CSS es la piel y la ropa que define cómo se ve una página web, permitiendo, de esa forma, una mejor presentación de la web.

JAVA SCRIPT

JavaScript (JS) es un lenguaje de programación que hace las páginas web interactivas y dinámicas. Si HTML es el esqueleto y CSS la apariencia, JavaScript es el músculo que da vida.

HTTP

HTTP (Hypertext Transfer Protocol) es el protocolo de comunicación que permite las transferencias de información en la World Wide Web. Es la base de cualquier intercambio de datos en Internet.
